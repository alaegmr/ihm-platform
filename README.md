## 💻 IHM Learning Platform  

A simple **web application** for the IHM (Human-Computer Interaction) course.  
Features: teacher-student communication, access to courses/TP/TD, and user authentication.  

### 🛠️ Technologies Used  
- HTML / CSS  
- JavaScript
